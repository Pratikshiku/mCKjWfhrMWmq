### 三、功能描述
系统分为（身份）：普通用户、垃圾分类管理员、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 垃圾分类
5. 垃圾图谱
6. 个人中心
7. 修改密码

**垃圾分类管理员功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 查看用户
6. 垃圾分类管理员
7. 垃圾分类
8. 垃圾类型
9. 垃圾图谱

**管理员功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 用户管理
5. 垃圾分类管理员
6. 垃圾分类
7.垃圾类型
8. 垃圾图谱
